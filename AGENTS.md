# Reglas del proyecto

## Memoria de sesion
- Antes de empezar cualquier tarea, leer `MEMORIA.md` para recuperar el contexto previo.
- Mientras se trabaja, mantener `MEMORIA.md` actualizado con avances, archivos modificados, pendientes y bloqueos.
- Al cerrar una tarea o dejar trabajo incompleto, guardar el estado final en `MEMORIA.md`.
- Si `MEMORIA.md` no refleja el estado real, corregirlo como parte del trabajo actual.

## Formato recomendado para `MEMORIA.md`
- Objetivo actual
- Ultimo avance
- Archivos tocados recientemente
- Pendientes
- Bloqueos
- Siguiente paso recomendado
